# React + TypeScript + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

## Getting Started

```bash
pnpm install
pnpm run dev
```

```bash
open http://localhost:5173
```

### Running Tests

```bash
# Run tests in watch mode
pnpm test

# Run tests once
pnpm test:run
```
